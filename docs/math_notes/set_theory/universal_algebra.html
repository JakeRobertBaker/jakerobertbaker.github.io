1 Universal Algebra

We denote the nullary set G^0 as \{ \emptyset \}.

Discussion on singleton set..

A structure is set equipped with finary operations.

A finary operation is a map from a set G to any of G^0, G^1=G, G^2 = G \times G, ... .

Definition 1 (Operational Type) An operational type is a pair (\Omega, \alpha) with

  • set of operation symbols \Omega
  • a map \alpha : \Omega \mapsto \mathbb{N} that assigns each \omega \in \Omega it’s arity \alpha(\omega).

Definition 2 (Operational Structure) An operational structure (\Omega, \alpha) is

  • a set A with an operational type (\Omega, \alpha)

  • a function \omega_A : A^{\alpha(\omega)} \mapsto A for each \omega \in \Omega

\omega_A is the interpretation of abstract symbol \omega in the structure A.

Definition 3 (Homomorphism) A map f:A\mapsto B is a homomorphism of \Omega structures A & B if

f(\omega_A (a_1,..., a_n)) = \omega_B (f(a_1),...,f(a_n))

where \alpha(\omega)=n and \forall a_1,...,a_n \in A

Definition 4 (The set of \Omega-terms in X) Let \Omega be an opertaional type and X be a set.

The set F_{\Omega}(X) of \Omega terms is defined inductively:

  1. If x \in X then x \in F_{Ω}(X)
  2. If w \in \Omega with \alpha(w)=n and t_1,...,t_n \in \Omega then wt_1...t_n \in F_{Ω}(X).

Note that this really the intersection of all operation structures that contain X.

Theorem 1 (F_\Omega(X) is a sigma structure)  

  1. The set of \Omega-terms in X, F_{\Omega}(X) has a structure.
  2. Given any \Omega structure F_\Omega(X) and any map f:X \mapsto A there exists a unique homomorphism \hat{f}:F_\Omega(X) \mapsto A

In other words F_\Omega(X) is the free structure generated by X

Definition 5 (Derived Operations) Let X_n= \lbrace x_1,...,x_n \rbrace, t \in F_{\Omega}(X_n), A be a \Omega structure.

The n-ary derived operation corresponding to t, t_A : A^n \rightarrow A is defined as

\forall \underline{a} \in A^n

t_A(\underline{a}) = \begin{cases} a_i &\text{if } t = x_i \in X_n \\ \omega_A \lparen t_1(\underline{a}) ,..., t_m(\underline{a}) \rparen &\text{if } t = \omega t_1 ... t_m, \text{ for } t_1,...,t_m \in F_{\Omega}(X_n), \alpha(\omega)=m \end{cases}

Note this really does just mean we keep applying the standard operations until we get to the X terms.

Therefore a derived operation really is just the map it represents

Example 1 (Derived Operationss are literally themselves)
t=mx_1 m x_2 x_3 has interpretation t_A : \underline{a} \mapsto m a_1 m a_2 a_3

When considering interpretation in F_{\Omega}(X_n) t_{F_{\Omega}(X_n)} (\underline{u}) = t[u_1/x_1, \dots, u_n/x_n]

Definition 6 (n-ary Equation) An n-ary equation in operational type (\Omega, \alpha) is an expression (s=t) for s,t \in F_{\Omega}(X_n)

Definition 7 (Satisfied) An n-ary equation is satisfied in operational structure A iff t_A \equiv s_A

Definition 8 (Algebraic Theory) An Algebraic Theory T=(\Omega, E) is an operational type (\Omega, \alpha) and set E of equations in \Omega

Definition 9 (Model) A model for an Algebraic Theory T=(\Omega, E) or T-algebra is an Operational Structure \Omega where all equations in E are satisfied.

Definition 10 (Derived Equations) Let E be a set of equations. The set \tilde{E} of derived equation satisfies:

  1. E \subset \tilde{E}

  2. \tilde{E} defines an equivalence relation s \sim_E t \iff (s=t) \in \tilde{E} on the set of terms F_{\Omega}(X)

    1. reflexive: (t=t) \in \tilde{E} \ \forall t \in F_{\Omega}(X)

    2. symetric: (s=t) \implies (t=s)

    3. transitive: (s=t),(t=u) \implies (s=u)

  3. Let s[u / x_i] denote replacing symbol x_i with u.

    1. (s=t) \in \tilde{E} \implies (s[u / x_i] = t[u / x_i])

    2. (s=t), u \in F_{\Omega}(X_n) \implies u[s/x_i] = u[t/x_i]

This is just the smallest subset of all possible equations (s=t) such that it is closed under a,b,c.

Definition 11 (Equivalence Classes of Equations on X) The equivalence classes of \sim_E on F_{\Omega}(X) is F_{(\Omega, E)}(X)

Theorem 2 (The set of equivalence classes satisfies the equations in E)  

  1. F_{(\Omega, E)}(X) inherits \Omega-structure from F_{\Omega}(X) and satisfies the equations in E.

  2. F_{(\Omega, E)}(X) is the free model generated by X.

    i.e For any map f: F_{\Omega}(X) \to (\Omega, E) \text{ model } A there is a unique homomorphism \tilde{f}: F_{(\Omega, E)}(X) \to A

Let [t] denote the equivalence class of t \in F_{\Omega}(X)

NoteProof (Proof of Theorem 2)

Proof (Proof of Theorem 2). Part I

In part(i) we need to show that this indeed an \Omega structure. This requires showing that the interpretation of \Omega operations in F_{\Omega}(X) respects the equivanece relation:

Consider operation \omega \in \Omega with \alpha(\omega)=n and symbols t_1,\dots, t_n and s_1, \dots, s_n where t_i \sim_{E} s_i \forall i \in \{1, \dots, n.

We need to show that the operations result is the same regardless of the representative picked for each equivalence class:

\begin{align*} \omega([t_1], \dots, [t_n]) &\overset{\text{definition}}{=} &\omega [t_1] \dots [t_n] \\ &\overset{\text{c.ii}}{\sim_E} &\omega [s_1] \dots [t_n] \\ \dots \\ &\overset{\text{c.ii}}{\sim_E} &\omega [s_1] \dots [s_n] \\ &\overset{\text{definition}}{=} &\omega([s_1],\dots,[s_n]) \end{align*}

Hence we have a valid interpretation that respects the equivalence relation.

The final part of (i) is to show that all the equations in E are satisfied.

Consider some equation (t=s) \in E, for it to be satisfied it must hold that their interpretations in F_{(\Omega, E)}(X) are equal.
